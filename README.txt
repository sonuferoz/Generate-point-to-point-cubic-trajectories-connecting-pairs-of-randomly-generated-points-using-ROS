To build and run the package, unzip the zip folder to the catkin workspace
source the setups and build the package using catkin_make
Once the package is build, launch the nodes using the launch file
Command to launch is "roslaunch ar_week5_txt cubic_traj_gen.launch". This is to be executed inside the package folder.
This will launch the four nodes, the rqt_plot and rqt_graph
